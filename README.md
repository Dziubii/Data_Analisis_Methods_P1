# Analiza skupień profili osobowości a wzorce spożycia substancji psychoaktywnych

## Opis projektu
Badanie korelacji między profilami psychologicznymi a skłonnością do sięgania po substancje psychoaktywne. Wykorzystano metody uczenia nienadzorowanego do identyfikacji naturalnych grup ryzyka.

## Autorzy
* **Karol Dziuba**
* **Aleksander Grzegrzułka**
* **Adam Grzywacz**

## Metodologia
* **Dane:** 1885 respondentów, zbiór z „UC Irvine Machine Learning Repository”.
* **Algorytmy:** Klasteryzacja K-Means oraz hierarchiczna metoda Warda.
* **Weryfikacja:** PCA, metoda łokcia, Silhouette Score oraz Adjusted Rand Index.
* **Zmienne:** Cechy osobowości NEO-FFI (neurotyczność, ekstrawersja, otwartość, ugodowość, sumienność) oraz impulsywność i poszukiwanie wrażeń.
